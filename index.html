Function Description

三種主要查詢功能，可以依營業時間查詢、相對距離查詢、快速搜尋

營業時間查詢

如果需要尋找特定時間營業的診所，歡迎使用「營業時間查詢」的功能唷！

相對距離查詢

如果想要查詢方圓一公里內的診所，歡迎使用「相對距離查詢」的功能唷！

快速搜尋

這裡可以快速查詢各診所的營業時間及詳細資料。
